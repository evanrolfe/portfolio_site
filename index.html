Freelance Portfolio

Customer Relationship Management System

A system that receives leads for the sale of solar panels from an API and distributes them to the appropriate solar panel installers who can login and view the leads they have received as well as pay for invoices which are generated automatically based on the number of leads received in a given billing period.

MOP! - Order a cleaner to your house

Quick online payment for customers - no registration required. Allows admins to manage their own cleaners, view order information, handle email subscriptions, set when cleaners are available in the calendar and more. Much of the code is covered by unit tests with phpunit 3.6 which were integrated to work with the MVC framework - fuelphp. Payment system uses the Paymill API. Graphic design by Joe Furr.

Bicycle routes tracker - joomla plugin

Allows users to map their on cycle tracks. It uses the google directions service API which means that the paths follow the road map between the points that users click.

Sales Manager and front end site

Enterprise scale application, this was developed for the owner of a business which facilitates the sale of time-shares in yachts around the world. It has a sale-tracking system built specifically for this business in order for them to keep track of their sales.

Garage sales tracker - wordpress plugin

Developed for a city's community web site, this allows users to enter the details of their garage sale, they simply enter their address and it using the google maps API to locate the coordinates and place a pin on the map for each garage sale.

Parent Teacher Conference Scheduler

This was developed for a school, parents usually want only 5-10min with each teacher but often times they have more than ten teachers to see in one session. The site allows parents to login and pick the times best suited for them. This saved the secretaries literally hundreds of hours of work, before this the secretaries had to take calls from each of the parents individually and work out what times were best one by one.
